# Book Library Collection

A React app showcasing a collection of books using reusable components, built with Vite.

## Components
- **BookCard**: Reusable component to display individual book information.

## Links
- **GitHub Repository:** https://github.com/umarmohammeds95-eng/My-Book-Collection.git
- **Live Website:** https://lighthearted-mandazi-3256ba.netlify.app

## Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Project Structure
```
book-library/
├─ index.html
├─ package.json
├─ vite.config.js
└─ src/
   ├─ main.jsx
   ├─ App.jsx
   ├─ index.css
   └─ components/
      └─ BookCard.jsx
```
